bitmap

分类于 Redis ， 数据结构

bitmap 位图，本质上就是一个很长的二进制数组

bitmap常用命令

  1. 设置某一位的值->setbit（初始化）

SETBIT key offset(位图中的第几位，offset 从0开始） value(只能是0或1)


  1. 获取某一位的值

GETBIT key offset


  1. 统计1的数量->bitcount

BITCOUNT key


  1. 按范围统计 → BITCOUNT key start end

统计区间范围内 bit=1 的数量。

这个start end 区间是按照字节来算的左闭右闭区间

  1. 多个bitmao之间的操作bitop

Redis 支持三种运算：

运算 含义
AND 与
OR 或
XOR 异或

求两天内任意一天在线
