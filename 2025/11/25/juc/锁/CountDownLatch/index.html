CountDownLatch

分类于 JUC ， 锁

CountDownLatch 是 Java 中用于实现线程同步的一种工具，它的作用是允许一个或多个线程等待其他线程完成某些操作后再继续执行。

一.基本概念

CountDownLatch 的名字可以直译为“倒计时锁”，它通过维护一个计数器来实现线程之间的等待。这个计数器会被其他线程逐步减少，直到变为零，表示所有等待的操作完成，调用 await() 方法的线程才会被释放，可以继续执行。

二.工作原理

  • CountDownLatch 在创建时会设置一个初始的计数值，这个计数值通常表示需要等待的事件或操作的数量。
  • 每当一个线程完成了它的任务，它就会调用 countDown() 方法将计数器减1。
  • 当计数器变为零时，所有调用 await() 方法的线程会被唤醒，继续执行。

三.常用方法

  1. 构造方法
    • CountDownLatch(int count): 创建一个 CountDownLatch 对象，初始化计数器为 count。
  2. **await()**** 方法**:
    • await(): 当前线程会等待，直到计数器为 0。线程会被阻塞，直到计数器减为零。
    • await(long timeout, TimeUnit unit): 当前线程会等待指定的时间，超时后会自动继续执行。
  3. **countDown()**** 方法**:
    • countDown(): 将计数器减1，表示一个线程完成了任务。
  4. **getCount()**** 方法**:
    • getCount(): 返回当前计数器的值。
