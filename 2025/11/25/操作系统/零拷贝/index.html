零拷贝

分类于 操作系统

什么是DMA

DMA（Direct Memory Access）允许 I/O 设备在 CPU 不参与数据搬运的情况下，直接访问内存，实现“设备 ↔ 内存”的高速传输。
CPU 只负责发起和结束，不负责搬运。

什么是零拷贝

零拷贝技术让数据在内核态内部直接流动，由 DMA 完成搬运，避免用户态与内核态之间的数据复制，从而减少 CPU 开销并提升 I/O 性能。

比如再进行网络io的时候，先来看没有零拷贝时的io方式

传统 I/O 中既有 4 次用户态与内核态的切换，也有 4 次数据拷贝；如果只是单纯搬运数据，这些操作会造成巨大开销，因此提升性能的关键就是减少上下文切换次数和内存拷贝次数。
