新switch

使用箭头->简化写法

旧写法

switch (x) {

case 1:

    foo();

    break;

case 2:

    bar();

    break;

}


新写法

switch (x) {

case 1 -> foo();

case 2 -> bar();

}


  • 无需 break
  • 不会穿透
  • 可读性高

switch 可以返回值

以前的switch 只能最为语句，不能返回值

现在可以

String result = switch (day) {

case 1 -> "Mon";

case 2 -> "Tue";

default -> "Other";

};


switch 本身是一个“表达式”了。

sealed

sealed 可以用来”限制谁能继承我”,避免类被随意扩展

核心语法

一个 sealed class：

  1. 用 sealed 修饰
  2. 必须指定 permits 列表
  3. 这些被允许继承的子类必须用：
    • final（不允许再继承）
    • sealed（继续限制）
    • non-sealed（完全开放）
